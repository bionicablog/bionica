# LQNTE — Lo Que Nadie Te Explica

Blog de divulgación ortopédica escrito por **Julián Urrea, Técnico Ortoprotésico**.

Artículos claros sobre ortopedia, pisada, movilidad y salud del aparato locomotor.  
Sin tecnicismos innecesarios. Para que entiendas qué te pasa y qué puedes hacer.

🌐 [bionicablog.github.io/bionica](https://bionicablog.github.io/bionica/)

## Sobre el proyecto

- 44 artículos publicados en 5 categorías: dolor · mayores · deporte · movilidad · consejos
- Aplicación de una sola página (HTML/CSS/JS puro, sin frameworks)
